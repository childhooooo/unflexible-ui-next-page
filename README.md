# Unflexible UI Legacy

Don't be flexible.

## Usage

### Installation

```bash
npm install unflexible-ui-legacy
```

### Optional settings

#### Wrap

```css
:root {
  --wrap-width-xl: 1240px;
  --wrap-width-l: 1030px;
  --wrap-width-m: 760px;
  --wrap-width-s: 470px;
  --wrap-width-xs: 94%;
}
```
